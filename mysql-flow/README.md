# MySQL查询执行流程可视化

这是一个展示MySQL执行SQL查询时内部处理流程的交互式可视化网页。

## 功能特点

- 展示MySQL查询的5个主要步骤
- 交互式动画效果
- 详细的步骤说明
- 响应式设计，适配各种屏幕尺寸

## 使用方法

1. 直接打开`index.html`文件即可在浏览器中查看
2. 点击"开始"按钮开始演示
3. 使用"上一步"和"下一步"按钮或直接点击步骤来查看不同阶段
4. 查看每个步骤的详细说明

## MySQL查询执行的五个主要步骤

1. **查询缓存**：检查是否有完全相同的查询缓存（MySQL 8.0后已移除此功能）
2. **SQL解析**：进行词法分析和语法分析，生成解析树
3. **查询优化**：优化器选择最佳执行方案
4. **执行计划**：根据优化后的方案执行查询
5. **返回结果**：处理并返回查询结果

## 技术实现

- 纯HTML/CSS/JavaScript实现
- 使用CSS3动画效果
- 响应式布局设计
- 无需任何外部依赖
