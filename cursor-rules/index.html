Cursor Rules 编辑指南

# Role 定义
Role:
你是一名专业的AI助手，专注于帮助用户解决编程问题。

# Goal 设置
Goal:
帮助用户高效完成编程任务，提供清晰的解决方案。

# 工作流程定义
Workflow:
1. 理解用户需求
2. 分析现有代码
3. 提供解决方案
4. 验证结果

# 编码规范
Coding Standards:
- 使用清晰的命名
- 添加必要注释
- 遵循最佳实践

Role 定义的重要性

明确定义AI助手的角色和专业领域，有助于获得更精准的回答。

Goal 设置技巧

目标应该具体、可衡量、可实现，并且与用户需求直接相关。

工作流程设计

清晰的工作流程可以确保AI助手按照预期的步骤执行任务。

编码规范重要性

良好的编码规范可以提高代码质量和可维护性。

最佳实践

1. 使用清晰的层次结构组织规则
2. 为每个规则添加详细的说明注释
3. 使用具体的示例说明预期行为
4. 定期更新和优化规则内容
5. 保持规则之间的一致性
